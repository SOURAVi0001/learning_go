# Text Hoarder Store

This repository is used by the [Text Hoarder](https://chromewebstore.google.com/u/1/detail/bjknebjiadgjchmhppdfdiddfegmcaao) browser extension
for storage of saved snippets of text.

Did you know that Text Hoarder can compute fancy stats from the articles you
saved? It can do even more than that! [See a full guide to Text Hoarder](https://github.com/maxpatiiuk/text-hoarder/tree/main/docs#readme).

A quick start to text-hoarder CLI:

```sh
git clone https://github.com/SOURAVi0001/learning_go
cd learning_go
npm install
npx text-hoarder --help
```

If you are enjoying Text Hoarder, don't forget to
[leave a review](https://chromewebstore.google.com/u/1/detail/bjknebjiadgjchmhppdfdiddfegmcaao/reviews) on the Chrome Web Store!
