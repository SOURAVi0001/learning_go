## Introduction to Go (Golang) 🚀
---

## 1\. Why use Go? (Go kyu use karte hain?)

## Simple Answer

Go ek **fast, simple aur scalable programming language** hai jo especially backend, cloud, APIs aur distributed systems ke liye bani hai.

Created by → Robert Griesemer, Rob Pike, and Ken Thompson at Google.

---

## Why Companies Love Go 🔥

| Feature | Meaning |
| --- | --- |
| Fast Compilation | Code bahut jaldi run/compile hota hai |
| Simple Syntax | Java/C++ se easy |
| Concurrency | Multiple tasks ek saath efficiently |
| Low Memory Usage | Servers pe efficient |
| Great for Backend | APIs, microservices ke liye best |
| Built-in Goroutines | Thread handling easy |
| Cross Platform | Windows/Mac/Linux sab pe |

## 5\. Go Commands 🛠️

## Most Important Commands

| Command | Use |
| --- | --- |
| `go run file.go` | Directly run |
| `go build` | Executable build |
| `go version` | Go version |
| `go env` | Environment variables |
| `go mod init` | New module create |
| `go get` | Package install |
| `go test` | Testing |
| `go fmt` | Format code |
| `go clean` | Cache/remove binaries |
| `go help` | Help |

---

## Very Important: go mod init

Modern Go projects modules use karte hain.
